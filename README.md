# Professor Assistant Program

## 📘 Project Overview
Our team is creating a **Professor Assistant Program** that helps professors generate exams from a question bank provided by the professor.  
This tool aims to save time, reduce manual effort, and ensure fair exam creation.

## 🚀 Features
- Import question banks provided by professors  
- Automatically generate exams based on selected criteria  
- Support for multiple exam formats (MCQs, short answers, etc.)  
- Easy-to-use interface for professors
